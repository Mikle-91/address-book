Тестируемое web приложение Address Book - адресная книга. Оно позволяет создавать контакты - карточки с контактными данными человека,
а так же создавать группы. Можно добавлять контакты в одну группу или в несколько групп(связи многие-ко-многим).
Данные хранятся в SQL базе, в трех таблицах:
-addressbook
-address_in_groups
-group_list

Функционал приложения:
-авторизация
-создание контакта
-удаление контакта
-редактирование контакта
-просмотр деталей контакта
-создание группы
-удаление группы
-редактирование группы
-добавление контакта в группу
-удаление контакта из группы
-на домашней странице отображается таблица контактов, содержащая First name, Last name, Address, All emails, All phones,
а так же кнопки просмотра/управления контактом;
-на странице с группами отображается таблица с названиями групп и кнопками управления.

Контакт содержит данные:
First name
Middle name/inital(s)
Last name
Nickname
Photo
Title
Company
Address
Home Phone
Mobile
Work Phone
Fax
E-mail
E-mail2
E-mail3
Birthday
Anniversary
Secondary
Address
Home
Notes

Группа содержит данные:
Group name
Group header (Logo)
Group footer (Comment)

Кроме тестов и вспомогательных файлов, разработан еще генератор тестовых данных для контактов и групп.
Он находится в папке generator. Сгенерированные данные сохраняются в папку data.
